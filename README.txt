شبکه سلامت — نسخه v2
=============================

فولدر شامل:
- index.html
- styles.css
- app.js
- assets/icon.png

۱) Google Sheet را «انتشار به وب» کنید (CSV)
--------------------------------------------
- از منوی شیت: File → Share → Publish to web
- در باکس بازشده: شیت پاسخ‌ها (Form Responses 1) را انتخاب کنید + فرمت CSV
- Publish → لینک می‌دهد که انتهایش `output=csv` است.

۲) لینک CSV را در app.js بگذارید
--------------------------------
داخل فایل `app.js` مقدار ثابت زیر را با لینک CSV جایگزین کنید:
```
const SHEET_CSV_URL = "https://docs.google.com/spreadsheets/d/XXXX/pub?output=csv";
```

۳) هدرهای پیشنهادی شیت (انعطاف‌پذیر)
-------------------------------------
این اسکریپت به‌صورت خودکار کلیدها را پیدا می‌کند، ولی این نام‌ها بهترین سازگاری را دارند:
- Timestamp
- عنوان پست  (یا: Title / پست)
- کپشن/توضیحات (یا: Caption / کپشن)
- هشتگ‌ها     (یا: Tags / برچسب‌ها)
- لینک        (Link)
- مدیا        (Media / آپلود / تصویر / ویدیو) — اختیاری

۴) نکات مهم درباره عکس/ویدیو
-----------------------------
- اگر لینک Google Drive باشد (مثل `/file/d/ID/view`) کد به‌صورت خودکار آن را به لینک مستقیم نمایش تبدیل می‌کند (`/uc?export=view&id=ID`).
- اگر لینک تصویر عمومی (jpg/png/webp) بدهید، کارت‌ها بندانگشتی را نمایش می‌دهند.
- اگر لینک mp4 عمومی بدهید، در گرید تصویر بندانگشتی نشان داده نمی‌شود (فعلاً ساده) اما دکمه «مشاهده» کار می‌کند.
- لینک‌های t.me (تلگرام) معمولاً فایل مستقیم نیستند؛ برای نمایش عکس بهتر است از Drive عمومی یا یک هاست تصویر استفاده کنید.

۵) استقرار
-----------
- این فولدر را در GitHub قرار دهید و به Vercel متصل کنید، یا به‌صورت Static روی هر هاستی آپلود کنید.
