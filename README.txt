شبکه سلامت — قالب سبک (HTML/CSS/JS)
======================================

1) داده‌ها از Google Sheet خوانده می‌شود.
   - در Google Sheets از مسیر: File → Share → **Publish to the web**
   - نوع را روی **CSV** بگذارید و لینک خروجی را کپی کنید.
   - در فایل `app.js` مقدار ثابت `SHEET_CSV_URL` را با آن لینک جایگزین کنید.

2) ساختار شیت (سطر اول هدرها):
   - Timestamp
   - دسته بندی
   - عنوان پست
   - کپشن/توضیحات
   - برچسب‌ها     (مثل: `#سلامت_روان #تغذیه`)
   - لینک          (لینک پست در تلگرام/سایت)
   - آپلود/تصویر/ویدیو (در صورت عمومی بودن، تصویر کوچک نمایش داده می‌شود)

3) صفحات:
   - خانه: دسته‌بندی‌ها (بر اساس اولین هشتگ شناخته‌شده)، لیست کارت‌ها + کپشن باز/بسته
   - اکسپلور: گرید مربعی شبیه اینستاگرام
   - جستجو: جستجو روی عنوان/کپشن/هشتگ
   - درباره: راهنما

4) استقرار:
   - فولدر را روی Vercel/Netlify/… آپلود کنید یا در GitHub پیاده‌سازی کنید.
